# CGM Forecast — Прогнозирование уровня глюкозы

Программная реализация курсовой работы по определению оптимального 
горизонта прогнозирования уровня глюкозы по данным непрерывного 
мониторинга (CGM). Пациент 588, датасет OhioT1DM.

## Модели
- ARIMA
- Метод Хольта (двухпараметрическое экспоненциальное сглаживание)
- XGBoost
- LSTM (PyTorch)

## Требования
- Python 3.11.9
- Зависимости: `pip install -r requirements.txt`

## Запуск

### Основной эксперимент
```bash
python run_experiment.py
```
Результаты сохраняются в папку `results/`: таблицы CSV и графики PNG.

### Анализ устойчивости
```bash
python run_sensitivity.py
```
Результаты сохраняются в папку `results_sensitivity/`.

## Данные
Файл `glucose_588.csv` содержит одномерный ряд глюкозы пациента 588.
Исходный XML-файл получен из датасета OhioT1DM и доступен по запросу:
Marling C., Bunescu R. The OhioT1DM Dataset, 2020.

## Структура проекта
```
cgm_forecast/
├── data/               — данные (glucose_588.csv)
├── src/
│   ├── config.py       — параметры эксперимента

│   ├── features.py     — формирование признаков
│   ├── metrics.py      — метрики качества

├── results/            — результаты (генерируется автоматически)
├── run_experiment.py   — основной эксперимент

└── requirements.txt
```
